Work Order: wo:asrs:599466
Subsystem: ignition
Source: ASRS (Aviation Safety Reporting System)

## Complaint
an airbus 319 was dispatched in non compliance with r engine 'a' ignition inop in conflict with the mel.

## Resolution
START FAULT WAS DETECTED DURING #2 ENG START WITH ECAM MESSAGE 'ENG #2 'A' IGNITION FAULT' 'RESTART' FADEC AUTOMATICALLY ABORTED, CLRED AND RESTARTED WITH ECAM MESSAGE 'AB' IGNITION ENG STARTED NORMAL. DURING AFTER START ECAM STATUS WAS CHK WITH 'NORMAL' NO FAULTS WAS NOTED! THINKING FADEC SAW SOMETHING IT DIDN'T LIKE THEN RESET OR FIXED ITSELF ON SECOND ATTEMPT, WE CONTINUED FLT TO ZZZ1. I DECIDED TO TALK TO MAINT ABOUT START WITH LOGBOOK ENTRY. IT WAS LATER DISCOVERED THAT 'A' SYS ENG HAD FAILED SOMEWHERE NOT SURE. ALSO FOUND OUT THAT ACFT WAS NOT TO BE DISPATCHED WITH A SYS INOP. SHOULD HAVE BEEN MORE EXPLICIT WITH LOGBOOK ENTRY.
