Work Order: wo:asrs:598402
Subsystem: ignition
Source: ASRS (Aviation Safety Reporting System)

## Complaint
a320 r engine 'b' ignition sys failed to start the engine ok on 'a' sys. no logbook entry made per maint ctl.

## Resolution
#2 ENG FAILED TO START. WE STARTED IT MANUALLY USING THE BOOK PROC AND CONTACTED MAINT. UNFORTUNATELY, WE DIDN'T SEE WHICH IGNITION SYS (A OR B) WAS USED DURING THE UNSUCCESSFUL START. THE MAINT ENG CTLR SUGGESTED WE START ENG #2 AGAIN USING AN AUTOSTART. WE STARTED THE #2 ENG AGAIN USING AN AUTOSTART AND THE 'A' IGNITION SYS. WE ASKED MAINT FOR A DEFERRAL OF #2 ENG 'B' IGNITION SYS. HE STATED WE EXPERIENCED AN ANOMALY IN IGNITION SYS AND NO DEFERRAL WAS REQUIRED. WE PROCEEDED TO ORD. ENRTE WE ENTERED AN ENGINEERING NOTE IN THE BOOK TO ALERT NEXT CREW IF IT WASN'T AN ANOMALY.
